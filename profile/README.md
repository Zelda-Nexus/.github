## About

We work at the intersection of **data engineering**, **knowledge graphs**, and **AI research** — applied to one of gaming's richest fictional universes. The goal: turn decades of Zelda lore into structured, queryable, machine-readable knowledge.

---

## Areas of Interest

| Domain | Focus |
|---|---|
| **Data Engineering** | Scalable pipelines, ETL, graph data modeling |
| **Knowledge Graphs** | Ontology design, RDF/SPARQL, entity linking |
| **Graph Neural Networks** | Node classification, link prediction, graph embeddings |
| **AI Research** | LLM integration, RAG systems, semantic search |
| **Open Source** | Community-driven datasets and tooling |

---

## Current Work

### Ocarina Nexus
> An open-source knowledge graph for *The Legend of Zelda: Ocarina of Time* — mapping characters, items, locations, quests and lore relationships into a structured graph database.
